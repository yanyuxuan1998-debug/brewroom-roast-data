# BrewRoom Roast Data

This repository stores your coffee roasting data managed by [BrewRoom](https://brewroom.dev).

- `brewroom.json` — Index of your beans, roast logs, and profiles
- `roasts/` — Roast log files (CSV/JSON from Kaffelogic)
- `profiles/` — Roast profile files (.kpro, CSV, JSON)

Do not manually edit `brewroom.json` unless you know what you're doing.
